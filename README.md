# Microcontrollers

Information, project templates, and example code for specific microcontrollers and evaluation boards. 

Also, debug interface and flash programming notes.

## Microcontrollers

* AVR
* 8051/8052
* STM8S/L
* STM32
* MSP430
* PIC

## Programmers

* USBasp
* USBtinyISP
* CH341A
* SP200S

## Debuggers

* SWD
* JTAG
* Serial interace adapters
