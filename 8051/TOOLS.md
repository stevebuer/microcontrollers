# 8051 Development Tools

A survey of 8051 development tools and IDEs.

## Command Line Assemblers

### AS31

Conveniently available in Debian package repository.

### a51 open source assembler

[Github Project](https://github.com/tmikov/a51)

### Microchip C51ASM

Windows command-line assembler supporting Microchip (formerly Atmel) AT89xxxx chips

## IDE for C and/or Assembly

### MCU 8051 IDE

Open source IDE

[SourceForge Project Page](https://sourceforge.net/projects/mcu8051ide)

### Keil uVision

Windows IDE is part of MDKv6

* C51 C compiler 
* A51 assembler
* debugger / simulator

* Supports all their C8051Fxxx chips
