# SP200S Serial Programmer

<img src="sp200.jpg">

Supports serially programmed flash chips and MCU with onboard flash.

* Atmel AT89, AVR, and EEPROMs
* SST
* ST
* Winbond
* Microchip

## Software

* Willem (Willar) software on Windows
* [Geepro on Linux](https://github.com/enriquebelarte/geepro)  TODO

## Supported Chips

A list of [supported chips](https://github.com/TomNisbet/asm85.git)

## Cables and Connectos

* 6pin ISP to development board for 8051
* 4 pin jimpers
