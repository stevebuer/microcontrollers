# AT24C256 EEPROM Module

Atmel AT24C256 (256KB) i2C EEPROM breakout board.

<img src="at25c256-module.jpg" width="200">

Pinout

* 1,2,3: A0, A1, A2 i2c address select
* 4: GND
* 5: SDA
* 6: SCL
* 7: WP (write protect)
* 8: VCC
