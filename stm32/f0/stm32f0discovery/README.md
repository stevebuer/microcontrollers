# STM32F0DISCOVERY

STM32F051R8 MCU

* 64k Flash
* 8k RAM

## LEDs and Buttons

* LD1 (red) 3.3v power on
* LD2 (red / green) USB comm
* LD3 (green) on PC9 
* LD4 (blue) on PC8
* Two buttons
