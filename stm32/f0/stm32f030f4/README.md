# STM32F030F4P6 Development Board

## MCU

* 48 Mhz CPU
* 16 KB Flash
* 4 KB RAM
* RTC, UART, I2C, API, ADC, GPIO

## Board

* 8 Mhz HSE crystal
* LED on PA4
