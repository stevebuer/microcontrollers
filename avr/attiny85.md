# ATtiny 85

8-bit AVR in 8-pin package. 

## General Specs

* 8K FLASH
* 812 bytes RAM
* 812 bytes EEPROM
* Hardware i2c / SPI
* 0 - 20 Mhz depending on supply voltage
* peripherals: ADC, 2 timers/counters, USI (serial), comparator
* 6 I/O lines
* 1.8V - 5.5V and 2.7V - 5.5V versions
* Lower power mode
