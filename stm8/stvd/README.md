# ST Visual Develop

These are examples and small projects using ST Visual Develop (STVD) on Windows 10.

The projects are further subdivided into pure ASM with ST Assembler and C with Cosmic C Compiler.

STM8 code editing support was added to Visual Studio Code since the STVD editing environment is dated.

## ST-ASM

STVD with ST Macro Assembler

## ST-COSMIC

STVD with Cosmic C Compiler

## Visual Studio Code

* Build Automation
* Flash and Debug launchers for STVD components
* Intellisense for C/SPL
* STM8 Assembly syntax highlighting
